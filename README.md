# sywac-style-bunker

> A chunky style for your [sywac](https://sywac.io/) CLI application.

![Screenshot](screenshot.png)

## Install

```console
$ npm install sywac
$ npm install sywac-style-bunker
```

## Usage

```js
const sywac = require('sywac');
const bunker = require('sywac-style-bunker');

sywac.style(bunker);
```
